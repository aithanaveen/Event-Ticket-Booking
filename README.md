# 🎫 Event Ticket Booking - Premium Edition

A high-end, modern React application for internal department event ticket booking. This project features a stunning **Dark Glassmorphism UI** with smooth animations and a premium dashboard feel.

## ✨ Key Features

- **🌌 Immersive UI**: Dark theme featuring background floating orbs and a dot-grid dashboard aesthetics.
- **🛡️ Glassmorphism**: High-quality `backdrop-filter` effects on all cards and interactive elements.
- **📊 Real-time Availability**: Visual progress bar tracking ticket sales with dynamic color-coding (Green → Yellow → Red).
- **📝 Advanced Booking Form**: 
  - Interactive **Ticket Stepper** (+/- controls).
  - **Live Total Calculation** as you adjust quantities.
  - **Inline Validation** with visual feedback (✓ and ⚠).
  - Simulated async processing with loading states.
- **🎉 Smart Confirmation**: 
  - Auto-generated **Unique Booking IDs** (e.g., TF2026-A3B2C1).
  - Animated success checkmark and **Confetti Burst** effect.
  - Full summary of booking details.
  - **🖨️ Print Receipt** support with specialized print-friendly styles.

## 🚀 Tech Stack

- **Core**: React 19 + Vite (for ultra-fast development)
- **Styling**: Vanilla CSS3 with Custom Design Tokens (CSS Variables)
- **Typography**: Inter (Google Fonts)
- **Icons**: Emoji & Custom SVG Graphics

## 🛠️ Performance & SEO

- Fully responsive design (Mobile, Tablet, Desktop).
- Optimized layout shifts using modern CSS grid/flexbox.
- Accessible heading structures and semantic HTML5.

## 📦 Local Setup

1. **Clone the Repo:**
   ```bash
   git clone https://github.com/aithanaveen/Event-Ticket-Booking.git
   cd Event-Ticket-Booking
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Run in Dev Mode:**
   ```bash
   npm run dev
   ```
   *The app will be available at `http://localhost:5173` (or the next available port).*

4. **Production Build:**
   ```bash
   npm run build
   ```
